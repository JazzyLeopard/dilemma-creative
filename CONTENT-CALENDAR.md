## Content Distribution

Each blog post should be repurposed:
1. **Instagram carousel** — Key takeaways as slides
2. **Instagram Reel** — 30-60 sec talking head summarizing the post
3. **LinkedIn post** — Shortened version for B2B visibility
4. **Email newsletter** — Monthly digest of latest posts

## SEO Checklist Per Post

- [ ] Primary keyword in H1, meta title, first 100 words
- [ ] Meta description (150-160 chars) with keyword + CTA
- [ ] 2-3 internal links (to services, niche pages, related posts)
- [ ] At least 1 image with descriptive alt text
- [ ] URL slug matches primary keyword
- [ ] Schema: BlogPosting with author, datePublished, image
- [ ] Open Graph image (1200x630)
